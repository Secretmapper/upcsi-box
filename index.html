CSI Box is a preconfigured Linux flavor that’s fast, easy to use, and hackable to the core. It’s the perfect development environment, made incredibly simple.

Plug and Play

Full featured, right out of the box (pun intended). Stop spending time setting up/installing tools, or making language/framework x work on Windows.

Powered by Ubuntu

Using ubuntu-server as its core, CSI Box allows you to seamlessly install ubuntu packages while leveraging content unique to DCS.

Incredibly Performant

CSI Box is designed from the ground up to be snappy and clutter-free, allowing even older hardware to run it. And we did this while keeping the interface easy to navigate and beautiful - it comes pre-installed with UI options and a couple of syntax themes in both dark and light colors.

By CSI For DCS

CSI Box is maintained by Computer Students just like you. We’ve experienced the common pain points in machine setups, and our goal is to reduce these. Having the same portable development environment allows everyone to work on the same page.
